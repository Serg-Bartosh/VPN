# VPN

Opensource vpn

### Run site

1. Download the repository

2. In terminal go to the folder where you installed the files
3. Run commands

```bash
sudo docker build -t appname .    
```

```bash
sudo docker run -p 8000:8000 appname
```

4. Open your browser
5. Enter http://0.0.0.0:8000/ in the address bar.
